1. Lambda problems:
WebSocket timeouts
Cold starts during calls
No GPU / ONNX friendly environment
Hard to maintain call state
Execution time limits (even with 15 mins)


| Aspect          | Azure STT   | AWS Transcribe |
| --------------- | ----------- | -------------- |
| Audio flow      | Push stream | WebSocket pull |
| SDK abstraction | High        | Low (manual)   |
| Cold start      | Needed      | Not needed     |
| Language detect | Easy        | Limited        |


Move logging → CloudWatch
Move Service Bus → SQS/EventBridge
Move TTS → Polly
Move STT → Transcribe
